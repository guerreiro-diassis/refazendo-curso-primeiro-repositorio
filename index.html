ciclo do estado dos arquivos

  1. untracked > git add > staged
  2. staged > git commit -m > unmodified
  3. unmodified > modificação > modified
  4. modified > git add > staged
  5. staged > git commit -m > unmodified

Passo a passo para preparar um arquivo do PC para o GitHub

  • git init: para inicializar a pasta com o git
  • git branch -M main: muda o nome do branch
  • git remote add origin: linkar nosso repositório local com o repositório remoto
  • git remote -v: para visualizar com qual repositório o nosso está linkado

Ccomandos e estados dos arquivos

  • git status: mostra o estado atual deles na pasta
  • git add .: manda todos os arquivos modificados para staged
  • git push -u origin main: envia para o site os arquivos commitados
